# Programador AO | API

Repositório da [API](http://github.com/Programador-AO/programador-ao-api) da plataforma Programador AO, construído pela comunidade

## Instalação

```bash
git clone https://github.com/programador-ao/programador-ao-api

cd programador-ao-api\src

npm install
```

## Executando o projeto

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Teste

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Criar a build

```bash
# build (pasta: dist)
npm run build

```
